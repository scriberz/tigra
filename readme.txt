Запуск проекта
Установите зависимости:

composer install
npm install

Создайте файл .env на основе .env.example и настройте подключение к базе данных.

Выполните миграции для создания таблиц в базе данных:

php artisan migrate

Соберите фронтенд ресурсы:

npm run dev

Запустите сервер:

php artisan serve

Теперь проект должен быть доступен по адресу http://localhost:8000, где можно протестировать функциональность списка элементов.
